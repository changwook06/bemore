# BeMore GESS 2026 Website

Publishable static website for BeMore, the KAIST GESS 2026 lifestyle continuity app concept.

## Local Preview

```bash
npm run start
```

Then open `http://localhost:5173`.

## Contents

- `index.html` - site content and page structure
- `styles.css` - responsive visual design
- `script.js` - prototype media tab interactions
- `assets/` - BeMore logos, app screenshots, and GIF demo flows
